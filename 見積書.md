## 1\. ご提案の概要と導入メリット

本見積もりは、弊社にて開発・実運用中の「楽天RMSレポート自動収集エンジン」を、貴社専用環境へカスタマイズ導入するものです。
ゼロからの新規開発ではなく、**検証済みの既存資産（資産価値：約120万円相当）** をパッケージとして導入することで、**低コストかつ短納期**での安定稼働を実現します。

### 導入による定量的メリット（ROI試算）

| 比較項目 | 従来の手作業・既存SaaS | **本システム導入** |
| :--- | :--- | :--- |

| **データ価値** | 過去データの蓄積が困難。<br>分析にタイムラグが発生。 | **自社DBへ無期限蓄積**。<br>リアルタイムに近いデータ分析・入札連携が可能。 |

## 3\. サービスレベル合意（SLA）および保証範囲

月額保守契約（推奨プラン）におけるサポート範囲を明確化いたします。

### 3.1 対応範囲（月額費用内）

  * **楽天RMSの仕様変更対応**:
    HTML構造の変更や、認証フローの軽微な変更（修正工数が1人日以内のもの）に対し、追加費用なしで修正プログラムを提供します。
  * **サーバー障害対応**:
    システムダウン検知後、原則**24時間以内**に復旧作業または原因報告を行います。
  * **技術Q\&A**:
    「取得したデータをBIツールで見たい」「スプレッドシートに連携したい」等の技術的なご相談に対し、月間5時間程度を目安にアドバイスを行います。

### 3.2 別途お見積りとなるケース

  * 楽天RMSの大規模な仕様変更（例：認証システムが根本から変更される等）
  * 取得データ項目の大幅な追加、または貴社独自のロジック追加実装

## 4\. 技術仕様・セキュリティ

本システムは「堅牢性」と「セキュリティ」を最優先に設計されています。

  * **自動化技術**: Python (FastAPI) + Playwright
      * 従来のSelenium等と比較し、高速かつ安定したブラウザ操作が可能です。
  * **実行環境**: Docker Container
      * 環境依存を排除し、将来的なサーバー移転やスケールアウトも容易です。
  * **認証情報管理**:
      * ID/パスワード等の機密情報は環境変数(.env)で厳格に管理し、ソースコード内には一切記述しません。
      * OAuth2 (Client Credentials) 対応により、n8n等の外部ツールとセキュアに連携可能です。
