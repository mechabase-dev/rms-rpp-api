## 付録: 技術仕様

### 使用技術
- **言語**: Python 3.11
- **フレームワーク**: FastAPI 0.104.0以上
- **ブラウザ自動化**: Playwright 1.35.0以上
- **コンテナ**: Docker, Docker Compose
- **Webサーバー**: Uvicorn

### API仕様
- **エンドポイント**: `GET /rpp-report`
- **パラメータ**: `date` (YYYY-MM-DD形式)
- **レスポンス**: CSVファイル
- **エラーハンドリング**: 400, 404, 500エラー対応

- 環境変数による認証情報管理
- .envファイルのGit除外
